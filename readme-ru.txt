== Description ==

Плагин дополнительной настройки WordPress для создания сайта социальной сети.

Позволяет реализовать: 

1) размещение инструментов входа и выхода пользователей в областях меню и виджетов 
   автономных сайтов социальной сети; 

2) функционирование инструментов подписки на обновления автономных сайтов участниками 
   социальной сети; 

3) улучшение инструментов управления и навигации по сайту (управление административной 
   панелью, пролистывание страниц, перенаправление страниц).

Ориентирован на работу в режиме WordPress Multisite совместно с плагином 
социальной сети BuddyPress. Может использоваться в обычных установках WordPress
для решения отдельных задач.

== Installation ==

1. Установить как и любой другой плагин.
2. Активировать для сети или для отдельного сайта.
3. Зайти (в консоли) "Настройки" -> "WP Customizer", выбрать нужные опции.
